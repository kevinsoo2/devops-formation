# DevOps Formation

Site de formation pour apprendre les bases du DevOps.

## Cours disponibles

- Ansible - Automatisation d'infrastructure
- Kubernetes - Orchestration de conteneurs
- Red Hat / RHEL - Administration système Linux

## Technologies

- Next.js 15
- Tailwind CSS
- TypeScript
- Deployed on Render

## Développement local

```bash
npm install
npm run dev
```

## Déploiement

Déploiement automatique sur Render à chaque push sur `main`.
